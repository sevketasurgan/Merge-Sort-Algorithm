## [16,21,11,8,12,22]  - Merge Sort Algorithm

1. `[16,21,11]` &emsp;//&emsp; `[8,12,22]`<br><br>
 
1. `[16]` - `[21,11]` // `[8]` - `[12,22]` <br><br>

1. `[16]` - `[21] [11]` // `[8]` - `[12] [22]`<br><br>

1. `[(16)]` - `[(11),21]`   // `[(8)]` - `[(12),22]`<br><br>

1. `[11,16,21]`  // `[8,12,22]`<br><br>

1. `[8,11,12,16,21,22]`
